Function template launch

boost::application::launch

Synopsis

// In header: <boost/application/launch.hpp>


template<typename ApplicationMode, typename Application, typename CustomType, 
         typename Context> 
  int launch(Application & app, CustomType & ct, Context & cxt, 
             system::error_code & ec);

Description

This file has a list of free function to launch an aplication using a specifyc mode.

In this version 2 flavors of application are supported:

  • common

  • server

The "launch" function can create any of this types, and launch will aready setup/add default control aspects to application context.

Two version of "launch" are available, the first receive a boost::system::error_code variable 'ec' that would be set to the result of the operation, and no thrown an exception of boost::system::system_error.

Creates a application, the ec ( boost::system::error_code& ec) will be set to the result of the operation, they should be tested for errors.

Parameters:

app

User application functor instance.

ct

The custom type of application, that will be passed as paramenter to application type istead a default custom type instance. User can customize this instance.

ec

Variable (boost::system::error_code) that will be set to the result of the operation.

Returns:

the return of application operator of user functor class that usually need be returned by main function to O.S.
