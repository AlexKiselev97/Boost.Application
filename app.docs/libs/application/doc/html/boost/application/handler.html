Class template handler

boost::application::handler — A class to be used by the classes that need execulte callback.

Synopsis

// In header: <boost/application/handler.hpp>

template<typename HandlerReturnType = bool> 
class handler {
public:
  // types
  typedef boost::function< HandlerReturnType(void) > callback;

  // construct/copy/destruct
  handler();
  handler(const callback &);
  ~handler();

  // public member functions
  void set(const callback &);
  bool get(callback *&);
  bool is_valid() const;
  handler & get_handler();

  // public static functions
  template<typename App, typename Handler> 
    static boost::function< HandlerReturnType(void) > 
    make_callback(App &, Handler);
};

Description

This callback is used on all handlers that application can define.

E.g.:

bool instace_aready_running();

All callbacks are executed by user selected "application mode" the return is a template that can be defined be the user.

In case of our sample, if your return true from instace_aready_running the application mechanism will continue, if your return false the application mechanism will exit.

handler public construct/copy/destruct

  1. handler();

    Constructs an void handler. 

  2. handler(const callback & cb);

    Constructs an handler.

  3. ~handler();

    Destruct an handler.

handler public member functions

  1. void set(const callback & cb);

    Set a callback to handler.

  2. bool get(callback *& cb);

    Get a callback pointer to futher execution. This method is used internaly by launch/application mode.

    Returns:

    true if callback pointer is valid. 

  3. bool is_valid() const;

    Check if a callback pointer to futher execution is valid.

    Returns:

    true if callback pointer is valid. 

  4. handler & get_handler();

    Retrieves handler.

    Returns:

    the handler instance.

handler public static functions

  1. template<typename App, typename Handler> 
  static boost::function< HandlerReturnType(void) > 
  make_callback(App & app, Handler h);

    Convenient static method to create a callback to be used by handler.

    e.g. to make a callback of method like this:

    struct myapp { bool instace_aready_running(); };

    myapp app;

    handler<bool>::make_parameter_callback(app, &myapp::instace_aready_running);

    Returns:

    a boost::function<HandlerReturnType (context&) > callback to be used on handler class.
