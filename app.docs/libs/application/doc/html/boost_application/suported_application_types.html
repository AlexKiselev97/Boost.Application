Suported Application Types

Common Application
Server Application

Boost.Application supports basically 2 flavors of applications that can be:

Common Application

This kind of application is a usual Interactive Terminal application.

Server Application

This kind of application generates a Service (Windows), or a background process/Daemon (Unix).

Note

Both versions can be extended using the plugin system provided by library (see 'Extending Application')

Other types of applications can be added in the future, e.g. Apache Module, IIS Module, FastCGI.
