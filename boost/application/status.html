Class status

boost::application::status — This aspect class is used internaly by applications types (modes)

Synopsis

// In header: <boost/application/aspects/status.hpp>


class status {
public:

  enum application_state { stopped = = 0, running, paused };
  // construct/copy/destruct
  status(application_state);

  // public member functions
  application_state state();
  void state(application_state);
  bool operator==(application_state) const;
  bool operator!=(application_state) const;
};

Description

Indicates the current state of application, 3 ways are possible:

  • application_stopped

  • application_running

  • application_paused (used only on server appliation on windows side)
